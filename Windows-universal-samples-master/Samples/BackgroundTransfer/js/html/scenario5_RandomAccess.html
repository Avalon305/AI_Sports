Random access downloads

When random access mode is enabled, apps can access file content and seek within that content while the download is in progress. If the data at the new seek position is not yet available, the transfer will prioritize that new position. This allows the app to alter the order in which remote file data is requested from the server, which is useful for streaming media scenarios.
