When communicating with a web service that requires a single language to be specified, one recommended technique for determining the language to send to the service is to create a resource which represents the url or query parameter to send to the service. This lets the resource management system make the appropriate choice for the best language to send.

This achieves appropriate language matching, consistent fallback across the user's preference list, and language consistency with the client side resources.
