Description:

The Trident Text controls expose properties to enable optimized views on the Touch Keyboard, namely InputScopes. You can change the input scope to one of many built-in options, some examples below. When you change the input scope, you can make it much faster and easier for users to enter data in your application.

Here is a selection of controls with an InputType set on them.

Touching into the edit control launches the Touch Keyboard with the appropriate keyboard layout enabled.

In Windows 10 on the PC, the touch keyboard does not display automatically if a hardware keyboard is connected or if the PC is in Desktop mode and "Automatically show the touch keyboard in windowed apps when there is no keyboard attached to your device" in Settings -> Devices -> Typing is disabled.
