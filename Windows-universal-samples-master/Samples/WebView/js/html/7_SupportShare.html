Apps with a WebView control can share the current web page in the WebView control via the Share contract. For this sample, the app is configured to share the current title and URL for the web page below, along with a screen capture of the WebView control. To share this information with another app, click the "Share" button.

Apps can also send the current web page in a WebView control to be opened in the default browser. To open the current page in a browser, click the "Launch in browser" button.

Finally, the same information can be placed on the clipboard to be used in other applications. To copy the current page's URL to clipboard, click the "Copy URL to clipboard" button.
